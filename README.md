# reineke.tipsforbitrix

Локальный модуль для 1C-Битрикс, который позволяет администраторам оставлять служебные заметки для конкретных страниц.

## Возможности

- привязка заметки к URL страницы;
- статусы заметок;
- цветовые метки;
- встроенный список всех заметок в админке;
- редактирование заметки прямо на странице.
- безопасная деинсталляция с возможностью сохранить данные.

## Структура

- `install/` — установка модуля, регистрация событий, копирование админских файлов;
- `admin/` — админские страницы и AJAX-обработчик;
- `lib/` — ORM-таблица, менеджер заметок и рендер UI;
- `lang/ru/` — русские языковые файлы.

## Установка

1. Разместить модуль в одной из папок:
   - `/local/modules/reineke.tipsforbitrix`
   - `/bitrix/modules/reineke.tipsforbitrix`
2. Открыть админку Битрикс и установить модуль `reineke.tipsforbitrix`.
3. После установки в меню настроек появится раздел "Заметки по страницам".
4. При удалении модуля можно сохранить таблицу с заметками для последующей повторной установки.

## Как это работает

- модуль создаёт таблицу `b_tips_for_bitrix_note`;
- регистрирует обработчик `main:OnProlog`;
- показывает блок заметки администратору на административной странице;
- сохраняет и читает данные через AJAX-обработчик и ORM.

## Требования

- 1C-Битрикс;
- PHP с поддержкой стандартного API Битрикс;
- права администратора для управления заметками.
